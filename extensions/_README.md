## Extensions Directory
IDE extensions and development environment integrations.
**Purpose**: VSCode/Cursor extensions, IDE configurations, development tools
**Actions**: Install extensions, configure IDE settings, integrate development tools
**Outputs**: Installation scripts, IDE configs → `vscode/`, Tool integrations → `tools/`
**Tools**: VyContext extension, VSCode marketplace, Open VSX Registry (Cursor)
**Installation**: 
- VS Code: Install from VS Code Marketplace
- Cursor: Install from Open VSX Registry (https://open-vsx.org/extension/vyges/vycontext)
- Use `install-vyges-extension.sh` for automatic installation
